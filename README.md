# Fastify Node API

## Description

Repository for learning and practicing NodeJS from the Rockeatseat Course.

## Technologies

- NodeJS
- Typescript
- Fastify (API)
- SQLite (Database)
- Knex (SQL Query Builder)

## Setup

Install Dependencies With

```bash
npm install
```

To run the project, use:

```bash
npm start
```

To run the project in developer mode, use:

```bash
npm run dev
```
